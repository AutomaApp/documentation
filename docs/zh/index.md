# 欢迎
这是 [Automa](https://www.automa.site) 的文档。Automa 是一个低代码/无代码的浏览器扩展，用于浏览器自动化。Automa 将帮助您自动执行所有操作，而不是手动输入、点击和从网站检索数据。
