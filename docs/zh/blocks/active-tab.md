# 活动标签块

将您当前所在的标签设置为活动标签。在此上下文中，__活动标签__ 是指需要标签的块，例如截图块将会在此标签上执行。

例如，当您想在当前网站上截图或执行 JavaScript 代码时，可以先使用活动标签，然后再连接其他块，如 [截图](/blocks/take-screenshot.md) 或 [JavaScript 代码块](/blocks/javascript-code.md)。
