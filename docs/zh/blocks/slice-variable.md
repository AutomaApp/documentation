# 切片变量块

提取变量值的一部分。

- **变量名称** <br>
	变量的名称。

- **起始索引** <br>
	开始提取的零基索引。
	如果起始索引为负，Automa 将从值的末尾开始提取。

- **结束索引** <br>
	要排除的第一个元素的索引。

**示例**
| 输入 | 起始索引 | 结束索引 | 输出 |
| --- | :---: | :---: | --- |
| `Hello world` | 0 | 5 | `Hello` |
| `Lorem ipsum` | 6 |  | `ipsum` |
| `Foo bar` | -3 | | `bar` |
| `[1, 2, 3, 4]` | 0 | 2 | `[1, 2]` |
| `[1, 2, 3, 4]` | -2 |  | `[3, 4]` |
