# 通知块

用于显示通知的块。

第一次使用此块时，点击块的编辑按钮，然后点击授予权限按钮。如果通知未显示，请尝试重启浏览器。如果您使用的是 Mac OS，您必须在系统设置中允许您使用的浏览器显示通知。

![mac 系统设置](https://s3.ap-southeast-1.amazonaws.com/automa-pub/i/2024/12/02/17zwfo-mc.png)

- **标题** <br>
	通知的标题。

- **消息** <br>
	通知的消息。

- **图标 URL** <br>
	通知的图标 URL。

- **图片 URL** <br>
	通知的图片 URL。
