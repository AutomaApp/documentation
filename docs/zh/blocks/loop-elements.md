# 循环元素块

遍历与选择器匹配的元素。此块与 [循环数据块](/blocks/loop-data.md) 的 "元素" 选项相同，但此块具有更多功能。

- **循环 ID** <br />
用于标识循环的 ID。当 [引用数据](#accessing-data) 或使用 [循环断点](/blocks/loop-breakpoint) 块时，请使用此 ID。

- **元素选择器** <br />
要遍历的元素的 [元素选择器](../workflow/element-selector.md)。

- **最大数据** <br />
设置要循环的元素限制。

- **加载更多元素** <br />
	加载更多元素的操作
	- **无**
	- **点击元素以加载更多**
	- **点击链接以加载更多**
	- **向下滚动以加载更多**
