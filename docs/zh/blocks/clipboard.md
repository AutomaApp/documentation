# 剪贴板块

从剪贴板获取复制的文本。

- **获取剪贴板数据**: 从剪贴板检索数据
	- **分配给变量** <br>
		是否将值分配到 [变量](../workflow/variables.md) 中。

	- **变量名称** <br>
		要分配值的变量名称。

	- **插入到表格** <br>
		是否将值插入到 [表格](../workflow/table.md) 中。

	- **选择列** <br>
		值将插入的列。

- **插入文本到剪贴板**: 将文本插入到剪贴板
	- **文本** <br>
		要插入到剪贴板的文本。
	- **复制页面上选定的文本**
