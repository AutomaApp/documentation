# 运行工作流

::: danger
在工作流运行时，请不要关闭 Automa 弹窗。因为关闭后工作流将无法运行。
:::

你可以通过打开 Automa 总览页并点击你想要运行的工作流上的播放图标 (<v-remixicon name="riPlayLine" />) 来运行工作流。

或者，你可以通过在页面上按 <kbd>Ctrl/⌘+Shift+E</kbd> 打开 Automa 快捷方式，然后选择工作流。
